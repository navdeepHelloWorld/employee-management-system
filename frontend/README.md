# Employee Management System (Frontend)

Vite + React + Tailwind demo for a simple employee task manager.

Features:
- Admin login creates tasks for employees
- Employee login views tasks and updates status (accept/complete/fail)
- Data persisted in localStorage (demo only)

Getting started:
1. npm install
2. npm run dev

Default users:
- Admin: admin@e.com / 123
- Employees: e1@e.com / 123, e2@e.com / 123

Notes:
- Local data is seeded once if missing; subsequent refreshes preserve changes.
